# Design Patterns

Este repositório foi criado com o objetivo de estudar e implementar os 23 padrões de projeto (Design Patterns) descritos no livro *Design Patterns: Elements of Reusable Object-Oriented Software (GoF)*.

Cada padrão é implementado de forma independente, buscando demonstrar sua estrutura, funcionamento e um exemplo prático de utilização.

## Objetivos

- Estudar os 23 padrões de projeto da GoF.
- Compreender quando utilizar cada padrão.
- Praticar conceitos de orientação a objetos.
- Explorar boas práticas de desenvolvimento em C#.

## Tecnologias

- C#
- .NET
